🔑 API Key Configuration
Your Anthropic API key is stored securely in Netlify Environment Variables — never in the browser. Set it once in your Netlify dashboard and JARVIS will always have access.
HOW TO SET YOUR API KEY IN NETLIFY:
1. Go to app.netlify.com → Your site
2. Site Configuration → Environment Variables
3. Click "Add a variable"
4. Key: ANTHROPIC_API_KEY
5. Value: your key from console.anthropic.com
6. Click Save → Trigger a redeploy

Get your key: console.anthropic.com → API Keys → Create Key
Cost: ~$0.003 per message

Deploys → Trigger deploy → Deploy site
